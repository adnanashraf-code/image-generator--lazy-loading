# 📸 Lazy Loading Image Gallery

A modern, high performance image gallery built using Vanilla JavaScript, featuring lazy loading, infinite scrolling, and real time search powered by the Pexels API.

This project demonstrates real world frontend concepts like API integration, performance optimization, and responsive UI design.

## ✨ Features

Real time image search  
Infinite scrolling auto pagination  
Lazy loading using IntersectionObserver  
Blur up image loading effect  
Skeleton loading UI  
Modal preview full size image view
